ONLY ONE KNIGHT WILL RISE

FIGHT, COMPETE, AND PRIVE YOUR DOMINANCE

In this competitive multiplayer experience, you and your friends become knights battling for glory. Use your skills, outsmart your opponents, and fight your way to victory in a world where only the strongest can win.

Target Audience

The game is designed for players who enjoy fast-paced online action and adventure, with a strong focus on competition and teamwork.

  • Ages 15–30
  • Fans of PvP gameplay
  • Enjoy team-based strategies
  • Prefer quick matches
  • Like pixel art & medieval themes

Built On

React

Used to build the game interface, allowing dynamic and interactive gameplay elements that update in real time.

Node.js

Powers the backend, handling game logic, player connections, and server-side processes efficiently.

Express.js

Manages the server and API routes, making communication between the frontend and backend smooth and organized.

MongoDB

Stores player data, scores, and game progress in a flexible and scalable database.
